## 🎯 Common Tasks

### Add More Products
1. Open `src/App.jsx`
2. Find the `products` array
3. Add new product object
4. Refresh browser ✓

### Change Colors
1. Open `src/App.css`
2. Find `:root { --primary-color: ... }`
3. Change color values
4. Refresh browser ✓

### Modify Filters
1. Open `src/components/FilterSidebar.jsx`
2. Change the `categories` array
3. Update filter logic in App.jsx if needed
4. Refresh browser ✓

### Deploy to Production
1. Run `npm run build`
2. Upload `dist/` folder to hosting
3. Website is live ✓
